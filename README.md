# AI NPC Learning Game

A browser-based prototype where players teach an NPC by choosing training actions, watching its knowledge grow, and testing whether it can solve new situations.

## Play

Open `index.html` in a browser.

## Current Features

- Train the NPC in logic, empathy, and exploration.
- Watch the NPC profile and learning stats update live.
- Run scenario tests to see how training affects NPC responses.
- Reset the simulation and try a different strategy.
